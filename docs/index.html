Configuration

The macros defined inside the configuration.h file are used to control the behaviour of the compilation and of the program.

  • PROGRAM_RP_ALLOCATOR This is a custom memory allocator, enabled by default. Set it to false if the program isn't working as expected.
  • PROGRAM_OUTPUT_COLORS This allows the program to use terminal colors, enabled by default. Set it to false if you are seeing random stuff on the terminal.
  • PROGRAM_SOLVER_TIMEOUT This is the timeout of the solver, set by default to 60 seconds.
    Warning
    Please adjust the settings based on your system!!!

Installation

In the provided files there is a shell script called compilation.sh

./compilation.sh

Or manually compile with cmake.

mkdir release && cd release
cmake -S ../ .. && make

A new folder called release will be created. You can now execute the executable.

./snake --file ../lab.txt

Arguments

The executable can be run with some arguments, to inject mazes or to trigger certain modes.

These are the arguments:

  • –file <path> Parses a maze from the specified file.
  • –generate <width> <height> Generates a maze and uses it in the game.
  • –challenge Runs the challenge mode,
    See also
    game_mode
