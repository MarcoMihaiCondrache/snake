◆ PROGRAM_RP_ALLOCATOR

#define PROGRAM_RP_ALLOCATOR   true

This macro determinate which memory allocator should be used. When set to true it will use a custom allocator, but if it isn't supported by your platform set it to false and the native allocator will be used.
