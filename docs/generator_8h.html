void generator_create ( maze_t *  m)

Function that will generate a maze and inject it in the provided param.

Will generate a maze based on the size provided by the pointer to the maze object.

The maze is generated with the recursive backtracking algorithm and then filled with random points and other game relative objects.

Note
The size of the maze must be odd, if even the function will decrease the size by 1 to make it odd.
Parameters
mPointer to maze object.
